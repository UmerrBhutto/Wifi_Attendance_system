# Automatic Attendance System Using Office Wi-Fi

An automated attendance management system built with Python that marks employee attendance when their registered devices connect to the office Wi-Fi network. The system records attendance in real time, reducing manual effort and improving accuracy.

## 🚀 Features

* Automatic attendance marking via Wi-Fi detection
* Real-time check-in with date and timestamp
* SQL database for secure and scalable storage
* Attendance history and reporting
* Reduced manual errors and HR workload

## 🛠️ Tech Stack

* **Python**
* **SQL Database**
* **Wi-Fi Network Scanning**

## 🎯 Applications

Suitable for offices, corporate environments, co-working spaces, and educational institutions requiring automated attendance tracking.
